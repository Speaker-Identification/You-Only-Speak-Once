You Only Speak Once

How to use this demo?

Expand / Collapse >

This webapp for demo is brittle with absolutely no thoughts given to UX. Read the instructions to use this demo!

Register

  1. select register in the "Choose Action" dropdown, fill in a username
  2. press record and start saying something. (microphone recommended). When done press submit
  3. Click, submit, you will get a successful register message

Authenticate

  1. Reload page to clear recording
  2. select login in the "Choose Action" dropdown, fill in a registered username
  3. press record and start saying something. (microphone recommended). When done press submit
  4. Click, submit, you will get a message saying if the voice is authenticated

You can say anything in any language. The "Read this" section is a random wiki article to help if you don't know what to say. While registering, a recording of 10-15 second is recommended. While authenticating a recording of 3-8 second is recommended.

Format: start recording to see sample rate
